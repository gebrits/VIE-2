Flickr

Drag'n'Drop semantic entities on this widget to trigger a photo search on Flickr.

The widget uses VIE^2 to automatically enrich the given entity and performs an entity-centered search.

This means that the way, the widget searches strongly depends on the type of the entity. When dropping an object that represents a place on the widget, a geo-location-driven search is started, whereas the search for persons uses their name.
